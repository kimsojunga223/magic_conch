# 마법의 소라고동🐚

### ⭐ 기획의도 ⭐
일기는 그날 있었던 일을 기록하기 위해 작성하기도 하지만 감정을 기록하기 위해서도 쓰인다. 
흔히 일기를 기록하는 수단인 글에서는 그 당시의 감정을 생생히 느끼기 어렵다. 
반면 음성은 그 사람의 감정을 직관적으로 느끼기 좋은 수단이다. 따라서 음성으로 하루를 기록할 수 있는 프로그램을 개발하고자 한다.

### 😶‍🌫️ 주요기능 😶‍🌫️
- 회원관리 및 로그인
- 달력으로 감정 표현
- 일기 음성 녹음
- 일기 음성 텍스트 변환
- 일기 태그 설정
- 커뮤니티
